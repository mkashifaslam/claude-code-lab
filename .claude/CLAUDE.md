# Claude Code Lab

This is a learning workspace for exploring Claude Code features before applying them to real projects.

## Purpose

Experiment with and document Claude Code features:

- CLAUDE.md and memory system
- Skills (custom slash commands)
- Hooks (lifecycle automation)
- MCP servers
- Subagents
- Settings and permissions
- Claude API / Agent SDK
- Scheduling and automation
- Agent teams

## Lab Conventions

- Each feature gets its own folder under `experiments/`
- Add a `README.md` to each experiment folder documenting findings
- Keep experiments focused and minimal — prefer clarity to completeness
- When something interesting is discovered, note it here or in memory

## Notes

- This is a sandbox — breaking things is fine
- Prefer real implementations over mocks to see true behavior
